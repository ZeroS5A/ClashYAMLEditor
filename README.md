# Clash / Meta 配置可视化编辑器

这是一个基于 React 和 Tailwind CSS 构建的**纯前端、单文件** Clash/Meta (mihomo) 配置文件可视化编辑器。它允许用户在浏览器中直观地导入、修改、管理和导出复杂的 YAML 配置，彻底告别手动编写容易出错的 YAML 语法。

本项目以**高性能、重交互、纯本地**为核心目标，内置了智能的分享链接解析器以及对海量规则集（如 ACL4SSR）的深度集成。

## ✨ 核心特性

### 📦 1. 智能导入与导出

- **多方式导入**：支持直接上传本地的 `.yaml` / `.yml` 文件，或直接粘贴 YAML 文本内容。
- **所见即所得**：对配置进行的所有可视化修改，随时可以通过“导出配置”面板实时预览生成的 YAML 源码。
- **一键导出**：支持一键复制到剪贴板，或直接下载为 `config.yaml` 文件。

### 🚀 2. 强大的节点 (Proxies) 管理

- **多协议支持**：支持 `ss`, `vmess`, `vless`, `trojan`, `hysteria2`, `tuic`, `wireguard` 等几乎所有主流/新兴协议的参数编辑。
- **万能链接解析**：内置高健壮性的 Base64 解码与 URI 解析引擎。只需粘贴分享链接（如 `vmess://`, `vless://`, `trojan://`...），即可自动提取 `ws-opts`, `grpc-opts`, `alpn`, `sni`, `client-fingerprint` 等复杂进阶参数，无损还原节点配置。
- **动态参数保留**：编辑时遇到未知的自定义参数，不会被丢弃，而是作为额外参数保留并支持修改。

### 🗂️ 3. 策略组 (Proxy Groups) 拖拽布局

- **丰富类型**：支持建立 `select` (手动), `url-test` (测速), `fallback` (故障转移), `load-balance` (负载均衡), `relay` (链式) 策略组。
- **丝滑交互**：无论是调整策略组在文件中的顺序，还是调整策略组内**包含节点的上下级顺序**，都全面支持**拖拽 (Drag & Drop)** 操作。

### 📚 4. 规则集 (Rule Providers) 深度集成

- **内置模板**：自带常用的 MetaCubeX (`mrs` 二进制格式) 与 ACL4SSR 基础模板，如 YouTube、国内 IP 等，一键生成定义。
- **ACL4SSR 动态联想**：无需查阅 GitHub，下拉框内置了搜索功能。通过 jsDelivr CDN 动态且稳定地拉取 ACL4SSR `master` 分支的规则列表，输入 `Amazon` 或 `BanAD` 即可一键补全 URL、Path 及格式配置。
- **策略组自动绑定**：添加规则集时可直接在底部选择“关联策略组”，系统将自动为您在路由规则中生成一条对应的 `RULE-SET` 记录。

### 🚦 5. 路由规则 (Rules) 智能管理

- **海量规则不卡顿**：针对包含上千条规则的复杂配置，底层采用虚拟化增量渲染机制，保证页面滚动与搜索的绝对流畅。
- **智能插入**：新增规则时可一键选择**“优先”**(插入到最前) 或**“扩展”**(智能插入到底部 `MATCH` 兜底规则之前)。
- **拖拽与检索**：支持对规则进行全文本/类型搜索。随时可以通过拖拽或置顶/置底快捷键调整规则的匹配优先级。

### ⚙️ 6. 基础通用设置

- 快速修改 `port`, `socks-port`, `mixed-port` 等监听端口。
- 一键切换 `allow-lan` (允许局域网)、`mode` (运行模式) 和 `log-level` (日志级别)。

## 🛠️ 技术栈

- **核心框架**: React 18
- **样式方案**: Tailwind CSS
- **图标库**: `lucide-react`
- **YAML 解析**: `js-yaml`
- **构建部署**: 零依赖单文件组件 (`ClashEditor.jsx`)，可无缝嵌入任何 React 宿主环境。

## 🔒 隐私与安全说明

本编辑器是**完全纯粹的前端应用**：

1. 您上传或粘贴的 YAML 配置文件**只会在您当前的浏览器内存中解析和处理**。
1. 没有任何后端服务器，**不会保存、上传、或分析您的任何节点（Proxy）及密码信息**。
1. （仅在规则集面板搜索时，会向 jsDelivr 请求公开的 GitHub 目录列表数据）。

## 💡 开发与扩展说明

- **新增协议字段**：可在代码顶部的 `PROTOCOL_SCHEMAS` 常量中轻松配置新协议的专属表单字段。
- **新增常用规则集**：修改 `RULE_PROVIDER_TEMPLATES` 和 `COMMON_ACL_FALLBACKS` 即可追加内置一键模板。
- 由于使用了浏览器原生的 `document.execCommand('copy')` 及 `FileReader`，本应用在标准的现代浏览器和内嵌 Webview (IFrame) 中均有出色的兼容性。
